What is the Clipboard API?

Provides read and write access to the operation system's clipboard.

Why should you use the Clipboard API?

While many users are familiar with copy/paste commands via a menu or keyboard shortcut, this API allows your page or application to provide intuitive options for getting content in and out of the clipboard.
