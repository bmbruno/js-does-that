What is the Gamepad API?

Allows the browser to read input from an attached gamepad device. This includes USB and Bluetooth devices.

Why should you use Gamepad API?

Beyond the obvious use cases of JavaScript-based games, the Gamepad API allows for alternate interface options for web apps.

Permission Required?

No.
