What is the Visibility API?

Determines if the current browser tab is visible to the user. The page is considered hidden when: the user changes browser tabs, minimizes the browser, or covers the browser with another window.

Why should you use the Visibility API?

If your webpage or app has interactive elements (music, video, game, etc.) or long-running process, you can pause actions when the page is not visible. This can ensure a good user experience or save system resources when your page isn't active.

Permission Required?

Yes, explicit user permission required.
