What is canvas?

The Canvas API and associated DOM elements provides primitive 2D and 3D drawing capabilities.

When and why should you use canvas?

The Canvas API is useful for drawing and animating complex graphics in a browser with a relatively simple set of APIs. Canvas is great for drawing complex charts, graphs, shapes, UIs, and animations, including JS-based games.

This contrasts with the more complex WebGL API, which provides hardware accelerated graphics rendering at the expense of implementation simplicity.
