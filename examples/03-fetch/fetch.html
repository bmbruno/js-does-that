What is the Fetch API?

The fetch() method provides functionality for fetching resources. It is available in both window and worker contexts (as well as Node environments). Fetch uses Request and Response objects for most interactions.

Why should you use the Fetch API?

Still using jQuery's ajax() method or XMLHttpRequest? Fetch is the universal replacement for handling GET and POST requests.

Permission Required?
