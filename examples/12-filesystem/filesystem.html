What is the File System API?

Provides access to read and write files on the local system.

NOTE: complete read/write functionality to the local file system are possible when combined with the File API and File System Access API.

Why should you use the File System API?

For web applications, access to local files can provide flexible options for storing content on a user's device for processing or interaction (for example, offline videos or other media).

Permission Required?
