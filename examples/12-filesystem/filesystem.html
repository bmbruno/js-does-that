What is the File System API?

Provides access to read and write files on the local system. Files written to disk are stored in a sandboxed area known as "origin private file system".

NOTE: this contrasts with the File API, which works locally in JavaScript stricly on files provided by the user (via an <input> control, for example).

NOTE: the File System Access API provides direct access to the entire local file system with a different API that explicitly requires user permission.

Why should you use the File System API?

For web applications, access to local files can provide flexible options for storing content on a user's device for processing or interaction (for example, offline videos or other media).
