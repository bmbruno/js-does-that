What is the Ink API?

Stylus, click, and touch events are normally captured via the browser JavaScript event loop. The Ink API is a performance-focused option that interfaces directly with the operating system's compositing layers. This ensures smooth, high-performance pen capture in a JavaScript application.

Why should you use the Ink API?

Use the Ink API to capture smooth pen or touch input from a user's device.
