What is the Idle Detection API?

With user permission, this API will monitor the current user's session for activity (based on session inputs) and screen locked status.

A 'change' event fires off when a user's idle status is updated.

Why should you use Idle Detection API?

If a user interacts with a long-running application (i.e. chat) or high-performance application (i.e. game), a change in idle status can provide updates ("user away") or pause process ("game paused").

Permission Required?

Yes, explicit user permission required.
