JavaScript Does THAT!? Examples

As featured in JavaScript Does THAT!? by Brandon Bruno


This source code is available on GitHub.

Geolocation

Gets the current device's latitude/longitude coordinates.

Clipboard API

Copy & paste content to the OS clipboard via JavaScript.

Fetch API

Get and send data/content from remote servers.

Canvas

Provides primitive 2D and 3D drawing capabilities.

Notifications

Send browser- and device-native notifications from a website or webapp.

Web Workers

Provides background thread execution of JavaScript (in a separate file).

Vibration API

Vibrate a device using timed patterns.

Idle Detection API

Determines if a user is active or idle on a webpage.

Gamepad API

Allows the browser to read input from an attached gamepad device.

Page Visibility

Determines if the current browser tab is visible to the user.

Background Fetch API

Allows large or long-running downloads to run in the background.

File System API

Provides access to read and write files on the local system.

IndexDB API

Provides a transactional database for storing large amounts of structured data on the client.
